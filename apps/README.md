# FEDAN Rate Alert Bot

A Python automation project that:

- Scrapes foreign exchange rates from FEDAN
- Detects rate changes
- Sends alerts through Viber
- Runs automatically using GitHub Actions

## Tech Stack

- Python
- BeautifulSoup
- Requests
- Viber Bot API
- GitHub Actions

## Features

- Automatic FEDAN scraping
- Change detection
- Scheduled execution
- Viber notifications
- Lightweight JSON/SQLite storage
